SEKAI{...} Flag Challenge

Cracking a Mathematical CTF Puzzle with Python

By Brahmi Sarah Kaouther · June 2026 · 6 min read

TL;DR: Solved a CTF challenge where the flag validation uses a regex pattern and a modular arithmetic condition. The key insight was recognizing that ~(6+~7)**67 = 2^67 - 1 and leveraging the cyclic property of powers of 2 modulo 2^67 - 1. The final flag has exactly 67 characters of '6' and '7' and satisfies the divisibility condition.

Challenge Overview

This challenge presents a Python assert statement that validates a flag string. The flag must satisfy two constraints:

  1. A regex pattern requiring exactly 67 characters consisting only of '6' and '7' inside SEKAI{...}
  2. A complex mathematical divisibility condition

Understanding the Assert Statement

challenge.py ~
__import__('re').match('SEKAI{[67]{67}}$', flag:=input()) and not int.from_bytes(flag.encode()) % ~(6+~7)**67

Breaking Down the Regex Pattern

The regex SEKAI{[67]{67}}$ means:

  • Must start with SEKAI{
  • Must have exactly 67 characters consisting only of '6' and '7'
  • Must end with }

Decoding the Mathematical Expression

Let's simplify ~(6+~7)**67 step by step, noting Python's operator precedence:

1. ~7 = -(7+1) = -8
2. 6 + ~7 = 6 + (-8) = -2
3. (-2)67 = -147573952589676412928
4. ~(-267) = -(-147573952589676412928 + 1) = 147573952589676412927

Notice that 267 = 147573952589676412928, so ~(6+~7)**67 = 267 - 1.

The condition not X % Y means X must be perfectly divisible by Y. Therefore:

int.from_bytes(flag.encode()) % (267 - 1) == 0

Mathematical Analysis

Flag Structure

The flag is 74 bytes total:

  • SEKAI{: 6 bytes (0x53, 0x45, 0x4b, 0x41, 0x49, 0x7b)
  • Inner 67 characters: each is either '6' (0x36) or '7' (0x37)
  • }: 1 byte (0x7d)

Encoding as a Number

Using big-endian encoding, the total value can be expressed as:

Total = Prefix × 2544 + Inner × 28 + Suffix

Where:

  • Prefix = int.from_bytes(b'SEKAI{', 'big')
  • Suffix = 0x7d = 125
  • Inner = sum over 67 positions of (54 + x_i) × 2(8 × position), where x_i ∈ {0,1}

Simplifying the Modulo Condition

In modulo 267 - 1, we have 267 ≡ 1. This creates a cyclic property where exponents wrap around modulo 67.

For each position y from 1 to 67 (where y represents bytes from the end), the contribution is:

2(8y) mod (267 - 1) = 2((8y) mod 67)

Since 8 and 67 are coprime, (8y) mod 67 for y = 1..67 produces a perfect permutation of {0, 1, ..., 66}.

Therefore, the variable part is exactly the binary representation of some number between 0 and 267 - 1.

Solution Strategy

Step 1: Compute the Base Value

First, compute the constant value where all inner characters are '6':

solve.py ~
base_bytes = b'SEKAI{' + b'\x36' * 67 + b'}'
C = int.from_bytes(base_bytes, 'big')

Step 2: Find Required Remainder

We need:

C + variable_sum ≡ 0 (mod 267 - 1)

Therefore:

variable_sum ≡ -C (mod 267 - 1)
solve.py ~
N = 2**67 - 1
target_rem = (-C) % N

Step 3: Map Bits to Positions

Since each position y maps uniquely to a power of 2, we can read the bits of target_rem:

solve.py ~
exp_to_y = {}
for y in range(1, 68):
    eff_exp = (8 * y) % 67
    exp_to_y[eff_exp] = y

S = ['6'] * 67
for eff_exp in range(67):
    if (target_rem >> eff_exp) & 1:
        y = exp_to_y[eff_exp]
        i = 67 - y
        S[i] = '7'

flag = "SEKAI{" + "".join(S) + "}"

Complete Solution Script

solve.py ~
#!/usr/bin/env python3

def solve_flag():
    # Base value with all '6's
    base_bytes = b'SEKAI{' + b'\x36' * 67 + b'}'
    C = int.from_bytes(base_bytes, 'big')

    # Modulus
    N = 2**67 - 1

    # Required remainder to make it divisible
    target_rem = (-C) % N

    # Map each effective exponent back to its position
    exp_to_y = {}
    for y in range(1, 68):
        eff_exp = (8 * y) % 67
        exp_to_y[eff_exp] = y

    # Reconstruct the 67-character string
    S = ['6'] * 67
    for eff_exp in range(67):
        if (target_rem >> eff_exp) & 1:
            y = exp_to_y[eff_exp]
            i = 67 - y
            S[i] = '7'

    return "SEKAI{" + "".join(S) + "}"

if __name__ == "__main__":
    print(solve_flag())

Verification

The solution can be verified by running it through the original assert:

verify.py ~
flag = solve_flag()
assert __import__('re').match('SEKAI{[67]{67}}$', flag) and not int.from_bytes(flag.encode()) % ~(6+~7)**67
print("✅ Valid flag found!")

Key Insights

  • Bitwise NOT Precedence: The expression ~(6+~7)**67 evaluates to 2^67 - 1, not 1.
  • Modulo Cyclic Property: In modulo 2^67 - 1, powers of 2 wrap around every 67 steps.
  • Unique Mapping: Because gcd(8, 67) = 1, each position maps to a unique bit position.
  • Binary Representation: The problem reduces to finding the binary representation of a specific remainder.
