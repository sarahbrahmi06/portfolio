Project 1: AISM Challenge, semantic Similarity

Problem Statement

The AI Sentence Meaning (AISM) Challenge required building a model to determine whether two sentences have the same meaning. The dataset consisted of sentence pairs with binary labels (1 = similar meaning, 0 = different).

Challenge Constraints

  • No pretrained LLMs allowed
  • Notebook submission mandatory
  • F1 Score evaluation

Approach

1. Data Preprocessing

preprocessing.py ~
def preprocess_text(text):
    text = text.lower()
    text = re.sub(r'[^a-zA-Z\s]', '', text)
    tokens = word_tokenize(text)
    tokens = [t for t in tokens if t not in stopwords]
    tokens = [lemmatizer.lemmatize(t) for t in tokens]
    return tokens

2. Siamese Network Architecture

model.py ~
class SiameseNetwork(nn.Module):
    def __init__(self, vocab_size, embed_dim, hidden_dim):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, embed_dim)
        self.lstm = nn.LSTM(embed_dim, hidden_dim, batch_first=True)
        self.fc = nn.Linear(hidden_dim, 128)
    def forward(self, input1, input2):
        embedded1 = self.embedding(input1)
        embedded2 = self.embedding(input2)
        _, (h1, _) = self.lstm(embedded1)
        _, (h2, _) = self.lstm(embedded2)
        features1 = self.fc(h1.squeeze(0))
        features2 = self.fc(h2.squeeze(0))
        similarity = F.cosine_similarity(features1, features2, dim=1)
        return torch.sigmoid(similarity)

Project 2: Image Segmentation, UNET + EfficientNet-B3

Problem Statement

The goal was to develop a semantic segmentation model capable of accurately identifying and segmenting objects in medical and satellite imagery. The challenge required precise boundary detection with high IoU scores.

Dataset

  • Training: 661 images with ground truth masks
  • Validation: 150 images with ground truth masks
  • Resolution: 720 × 960 pixels
  • Target: Water bodies (flooded areas, rivers, lakes, puddles)

Approach

1. Architecture: UNET with EfficientNet-B3 Backbone

segmentation.py ~
class BoundaryAwareUNet(nn.Module):
    def __init__(self):
        super().__init__()
        self.backbone = EfficientNetB3(pretrained=True)
        self.unet = UNetDecoder()
        self.boundary_attn = BoundaryAwareAttention(64)
        self.final = nn.Conv2d(64, 1, 1)
    def forward(self, x):
        features = self.backbone(x)
        features = self.unet(features)
        enhanced = self.boundary_attn(features)
        mask = torch.sigmoid(self.final(enhanced))
        return mask

2. Boundary-Aware Loss Function

loss.py ~
class EdgeAwareLoss(nn.Module):
    def __init__(self, edge_weight=0.5):
        super().__init__()
        self.edge_weight = edge_weight
        self.sobel = SobelEdgeDetector()
    def forward(self, pred, target):
        bce_loss = F.binary_cross_entropy(pred, target)
        pred_edges = self.sobel(pred)
        target_edges = self.sobel(target)
        edge_loss = F.mse_loss(pred_edges, target_edges)
        return bce_loss + self.edge_weight * edge_loss
