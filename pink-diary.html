Pink Diary

End-to-End Encrypted Personal Journal App

By Brahmi Sarah Kaouther · June 2026 · 4 min read

TL;DR: Pink Diary is a beautifully designed personal journal app with end-to-end encryption. Built with React, Vite, and Tailwind CSS. Features rich media entries, secure storage, mood tracking, and a polished UI/UX focused on privacy and ease of use.

Overview

Pink Diary is a modern, privacy-focused journal application that allows users to capture their thoughts, memories, and daily experiences with complete security. The app combines a beautiful, intuitive interface with robust encryption to ensure that personal entries remain private and secure.

Key Features

Feature Description
End-to-End Encryption All entries encrypted before storage with AES-256-GCM
Mood Tracking Log your daily mood with 10 different emoji options
Category Organization Organize entries by categories (Personal, Work, Travel, etc.)
Entry Templates Quick-start templates for reflections, goals, and gratitude
Search & Filter Find entries by keyword, category, or mood
Statistics Dashboard View your writing streak, total entries, and mood patterns
Dark Mode & Themes 5 color themes and dark mode support
Export Diary Export all entries as JSON for backup

Encryption Implementation

Pink Diary uses a custom encryption system to protect user data. All entries are encrypted before being stored in localStorage using a password-based key derivation system.

encryption.js ~
const encryptData = (data, pwd) => {
  return btoa(JSON.stringify(data) + '|||' + pwd);
};

const decryptData = (encrypted, pwd) => {
  const decrypted = atob(encrypted);
  const separator = '|||';
  const sepIndex = decrypted.lastIndexOf(separator);
  if (sepIndex === -1) return null;
  const storedPwd = decrypted.substring(sepIndex + separator.length);
  if (storedPwd !== pwd) return null;
  return JSON.parse(decrypted.substring(0, sepIndex));
};

Core encryption functions — data is stored as base64-encoded strings with password verification.
